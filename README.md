# platform
开发平台
任何业务系统可以抽象出本质:表单+流程+报表
登陆、授权、单据、流程、报表。
基于一套稳健，成熟的开发平台，可以定制出任何复杂的业务系统：电商、O2O、CMS、CRM、BPM、MES、ERP、OA...
基于一套稳健的基础架构、监控系统、分布式、可扩展、高可用的服务组成。
此platform更偏向于业务架构，具备快速迭代开发产品、项目、充当轮子的作用。
###################
有如下的功能:
1、登陆
2、授权
3、接口验签
4、表单迅速迭代开发
5、流程可定制配置
6、报表可见即所得
7、支持横向扩展
###################
1、基于JDK1.8+
2、Spring
3、Mybatis
4、MySql
5、SpringBoot
6、Nginx
7、阿里云ELB
